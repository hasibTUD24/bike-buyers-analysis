Bike Buyers Analysis

Overview

This project analyzes a bike buyers dataset to understand customer behavior and identify factors that influence bike purchases. It uses data cleaning, exploration, and visualization to draw insights.

Dataset

The dataset contains customer information such as:

Age
Income
Marital Status
Occupation
Whether the customer bought a bike

The data was cleaned to remove duplicates and missing values.

Analysis

Explored customer demographics

Checked how income, age, and occupation relate to bike purchases

Created visualizations to identify trends

Visualizations

All charts and graphs are saved as PNG files in the Visualizations folder.

How to Use

Clone this repository:

git clone https://github.com/hasibTUD24/bike-buyers-analysis


Open the dataset in Excel or Python to explore.

View visualizations in the Visualizations folder.

Conclusion

The analysis helps understand which customers are more likely to buy bikes and can guide marketing decisions.
